# Diabetes Prediction Dashboard

An interactive Streamlit dashboard for training, comparing, and visualizing
multiple ML models (RandomForest, XGBoost, LightGBM, SVM, Logistic Regression)
for diabetes prediction.

## 🚀 Features
- Notebook-Mode (High accuracy, includes glyhb features)
- Safe-Mode (Leakage-free evaluation)
- Visual comparison (ROC, Confusion Matrix, Radar Charts)
- SHAP explanations for supported models
- Interactive prediction form

## 🔧 How to Run
```bash
pip install -r requirements.txt
streamlit run streamlit_diabetes_dashboard.py
